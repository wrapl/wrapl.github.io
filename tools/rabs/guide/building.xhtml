Building and Installation

Linux - From Source

Rabs can be easily built on Linux from source. The only dependencies are:
  1. Standard build tools (gcc, make, etc).
  2. The Hans-Boehm garbage collector, available in the standard repository of most distros as libgc-dev or similar.
  3. Sqlite3, available in the standard repository of most distros as libsqlite-dev or similar.
Once these are installed, building Rabs should just be a matter of running:
git clone https://github.com/wrapl/rabs cd rabs make sudo make install

Mac OS - Using Homebrew

Using Homebrew, Rabs can be installed and kept up to date easily from the Rabs tap.
brew install wrapl/rabs/rabs

Windows - From Source

Building on Windows using Mingw is similar to building on Linux.

Windows - Binaries

Binaries are now available for Windows at the Rabs releases page. These are built with Mingw but can be run without Mingw installed.
